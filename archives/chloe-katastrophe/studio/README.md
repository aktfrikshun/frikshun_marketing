# Chloe Studio CDK

This folder is the private production bible for creating Chloe Katastrophe consistently across images, video, music, TikTok, album visuals, and future tools.

## Studio Rule

The studio folder is **not** the public archive.

It contains production guidance, model references, prompt libraries, review workflows, and approved visual references for Chloe herself.

Archive artifacts, album covers, Gregor boards, family records, and public-facing lore belong outside this folder unless they are being used specifically as production references.

## Workflow

1. Generate candidate Chloe assets.
2. Review against the model canon.
3. Reject anything that drifts.
4. Promote approved assets into `reference-decks/approved/v1`.
5. Use approved references for future image/video generation.

Chloe should become more consistent over time, not reinvented every time.
