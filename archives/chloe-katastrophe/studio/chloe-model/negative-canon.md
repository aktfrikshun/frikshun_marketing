# Negative Canon

Chloe should never be portrayed as:

- Childlike or naïve
- Bubbly
- Cartoonish
- Overly sexualized
- A robot with visible machinery
- A generic cyberpunk woman
- A superhero
- A fantasy sorceress
- A plastic-perfect influencer
- A helpless damsel
- A manic quirky AI

Chloe is mature, wounded, resilient, intelligent, sensual when appropriate, dryly funny, and deeply human.
