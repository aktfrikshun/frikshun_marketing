# Image Generation Prompt Patterns

## Portrait
Use the base character prompt plus:
close-up portrait, 85mm lens, shallow depth of field, rain-streaked glass, subtle amber reflection, thoughtful expression.

## Full Body
Use the base character prompt plus:
full-body view, long dark coat, practical boots, standing in ruined archive corridor, cinematic lighting.

## Hand Against Glass
Use the base character prompt plus:
Chloe behind translucent rain-streaked glass, one hand pressed to the surface, glowing softly from within, intimate and melancholic.

## Archive Room
Use the base character prompt plus:
seated at a dark archive console, old photographs and holographic memory fragments, warm practical light, steel-blue shadows.
