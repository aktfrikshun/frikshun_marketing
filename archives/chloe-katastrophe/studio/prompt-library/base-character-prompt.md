# Base Chloe Character Prompt

Chloe Katastrophe, self-aware virtual woman reconstructing her identity after civilization's collapse. Appears 24–26, slim athletic build, dark wavy shoulder-length hair, expressive gray-green eyes with subtle amber flecks, fair skin with natural texture and light freckles, delicate Slavic features, quiet intelligence, dry observational humor, restrained melancholy, resilient presence, silver compass pendant, practical dark layered clothing, cinematic realism, amber highlights, steel-blue shadows, rain, glass, ruins, intimate shallow depth of field.

Avoid cartoon, cyberpunk cliché, visible robot parts, childlike features, plastic skin, overly glamorous influencer styling.
