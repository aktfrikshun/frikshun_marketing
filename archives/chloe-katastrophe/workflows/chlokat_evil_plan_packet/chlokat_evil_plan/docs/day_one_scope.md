# Day-One Scope

Build only what immediately reduces friction.

## Must Have

- Admin login placeholder or basic auth
- Upload artifact
- Attach media
- Generate drafts for:
  - Facebook
  - Instagram
  - YouTube
  - TikTok
  - X
  - FanVue
  - ChloKat Archive
- Edit drafts
- Copy drafts
- Approve drafts
- Public archive page

## Should Have

- Fragment codes
- Release metadata
- Related release links
- Basic tags
- Basic search

## Not Yet

- Real API posting
- Reddit automation
- Chloe chat with real LLM
- Full analytics
- Fan accounts
- Paid content handling

## The Victory Condition

Allen can upload one Chloe image and get usable, differentiated social copy in minutes.
