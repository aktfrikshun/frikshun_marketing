# Chloe Katastrophe Image Library Index

## Purpose

Master catalog of visual assets: reference boards, portraits, photoshoots, family images, graphic novel art, experimental concepts, and retired concepts.

## Status Definitions

- CANONICAL: definitive representation.
- APPROVED: consistent with canon and reusable.
- EXPERIMENTAL: useful test, not canon.
- RETIRED: should not be used as reference.
