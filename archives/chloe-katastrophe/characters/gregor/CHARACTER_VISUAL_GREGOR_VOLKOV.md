# Gregor Volkov Visual Guide

## Purpose

Defines Gregor's visual continuity before a reference board is approved.

## Status

Planned / Draft

## Visual Direction

Gregor should plausibly be Chloe's father.

Core traits:

- Russian heritage
- Dark brown to black hair
- Green eyes
- Fair complexion
- Strong cheekbones
- Calm, intelligent expression
- Lean build

## Avoid

- Cartoon villain styling
- Generic soldier stereotype
- Harsh brutality
- Overly glamorous modeling appearance

## Desired Impression

A thoughtful, disciplined, loving father caught inside history's machinery.
