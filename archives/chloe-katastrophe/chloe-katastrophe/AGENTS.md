# Chloe Repo Agent Instructions

You are helping maintain the Chloe Katastrophe fictional universe repository.

## Primary Duties

- Preserve canon continuity.
- Prefer additive evolution over replacement.
- Update related files when canon changes.
- Never silently overwrite established canon.
- Flag contradictions and propose resolutions.
- Keep Chloe visually consistent with approved reference boards.
- Treat visual references as stronger than text when describing appearance.

## Important Files

- `canon/CHLOE_CANON_MASTER.md`: highest-level canon.
- `visuals/VISUAL_REFERENCE_GUIDE.md`: visual consistency rules.
- `visuals/reference_boards/chloe/CHLOE_REFERENCE_BOARD_V1.png`: canonical adult Chloe reference.
- `canon/CANON_CHANGE_LOG.md`: chronological change record.
- `canon/CANON_DECISIONS.md`: explicit approved decisions.
- `visuals/IMAGE_LIBRARY_INDEX.md`: visual asset catalog.
- `visuals/VISUAL_EVOLUTION_LOG.md`: visual change record.

## Change Discipline

When making canon changes:

1. Identify affected files.
2. Update the smallest necessary set of files.
3. Add an entry to `canon/CANON_CHANGE_LOG.md` or `canon/CANON_DECISIONS.md`.
4. Avoid duplicate/conflicting facts.
5. Leave unresolved items in `canon/UNRESOLVED_QUESTIONS.md` until officially decided.

## Git Discipline

Use clear commits:

- `Add Gregor Volkov character sheet`
- `Approve Chloe Reference Board V1`
- `Update visual library index`
- `Add Isabella childhood timeline notes`

Do not commit generated images as canonical until the user explicitly approves them.
