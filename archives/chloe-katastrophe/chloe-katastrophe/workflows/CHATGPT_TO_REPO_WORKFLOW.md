# ChatGPT to Repo Workflow

## Purpose

Capture ideas from ChatGPT conversations and convert them into repo changes.

## Best Practice

Do not try to preserve every chat message. Preserve decisions, artifacts, and canon changes.

## Process

1. Ask ChatGPT for a canon update packet.
2. Paste or save that packet into `chat_exports/` if useful.
3. Use Cursor/Codex to apply the decisions to the repo.
4. Commit approved changes.

## Canon Update Packet Template

```markdown
# Canon Update Packet

Date:

Summary:

Approved Decisions:

Affected Files:

New Unresolved Questions:

Visual Assets:

Commit Message Suggestion:
```
