# Public Website Prototype

This folder contains a static public-facing site for the **Daughter of Echoes** / Chloe Katastrophe archive.

## Recommended Hosting

Primary recommendation: host under the FrikShun website.

Suggested paths:

- `/chloe`
- `/archive/chloe`
- `/daughter-of-echoes`

Why:

- FrikShun is canonically Chloe's archivist/collaborator.
- The repo remains the source of truth.
- Social profiles can link back to one canonical public archive.
- It supports the reality/fiction blur without making Facebook or SoundCloud the canon host.

## Other Options

- GitHub Pages: fastest and free.
- Netlify / Vercel: good for custom domain and static deployments.
- Facebook Page: useful for promotion, not ideal for canon.
- Notion: fast for drafting, less immersive long-term.

## Deployment

This is plain HTML/CSS. Upload the `site/` directory to any static host.

If deployed under a subpath, no build step is required.
