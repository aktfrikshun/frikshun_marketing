# Chloe Katastrophe Canon Repository

This repository is the source of truth for Chloe Katastrophe: canon, visual references, character sheets, story arcs, image prompts, and brand/worldbuilding notes.

## Working Model

- ChatGPT Project: creative workshop and high-level conversation.
- Local repo: canonical files and assets.
- Cursor/Codex: repo-native editing agent.
- Git: approval history and rollback system.

## Suggested Workflow

1. Discuss ideas with Chloe in ChatGPT, Cursor, or Codex.
2. Make edits in a feature branch.
3. Review diffs.
4. Commit approved canon changes.
5. Keep generated images in `visuals/working/` until approved.
6. Move approved images into `visuals/reference_boards/`, `visuals/approved_portraits/`, or `visuals/approved_photoshoots/`.
7. Update the relevant index and changelog.

## Canon Rule

Visual Reference Board V1 is the baseline for Chloe's adult appearance. Existing canon says approved visual references override text descriptions when conflicts occur.
